## ğŸŒŸ Pengenalan Proyek

### Apa itu Kediri Navigasi?

**Kediri Navigasi** adalah aplikasi web responsif dan progresif (PWA) yang berfungsi sebagai portal wisata resmi Kota Kediri, Jawa Timur. Aplikasi ini dirancang untuk memudahkan wisatawan domestik maupun mancanegara dalam mengeksplorasi destinasi wisata, kuliner, dan informasi penting lainnya tentang Kota Kediri.

Dengan mengintegrasikan teknologi mapping interaktif, informasi real-time, dan pengalaman pengguna yang intuitif, Kediri Navigasi menjadi panduan digital komprehensif yang menyediakan segala yang dibutuhkan untuk merencanakan dan menikmati perjalanan wisata ke Kota Kediri.

### Visi dan Misi

**Visi:**
Menjadi portal wisata digital terdepan yang memberikan informasi komprehensif dan akurat tentang destinasi, kuliner, dan budaya Kota Kediri kepada seluruh wisatawan dunia.

**Misi:**
- Menyediakan informasi wisata yang akurat, terlengkap, dan mudah diakses
- Meningkatkan visibilitas destinasi wisata Kota Kediri secara global
- Membantu wisatawan dalam merencanakan perjalanan yang sempurna
- Mendukung pengembangan ekonomi pariwisata lokal
- Membangun komunitas pecinta wisata Kediri yang solid

### Target Pengguna

- **Wisatawan Domestik:** Traveler dari berbagai daerah di Indonesia yang ingin mengeksplorasi Kota Kediri
- **Wisatawan Mancanegara:** Pengunjung internasional yang mencari informasi tentang destinasi di Jawa Timur
- **Peneliti & Akademisi:** Pelajar dan peneliti yang tertarik dengan sejarah, budaya, dan kuliner Kediri
- **Pengusaha Pariwisata:** Pelaku bisnis yang ingin mempromosikan produk dan layanan wisata mereka
- **Perencanaan Perjalanan:** Travel agent dan tour operator yang membutuhkan referensi destinasi

## ğŸš€ Panduan Instalasi

### Requirement Sistem

**Minimum:**
- Node.js 14.x atau lebih tinggi
- npm 6.x atau yarn 1.x
- Browser modern (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- 100MB free disk space

**Recommended:**
- Node.js 18.x LTS
- npm 8.x atau yarn 3.x
- 500MB free disk space
- Koneksi internet 5 Mbps minimum

### Step-by-Step Installation

#### 1. Clone Repository

\`\`\`bash
# Menggunakan HTTPS
git clone https://github.com/yourusername/kedir-navigasi.git

# Atau menggunakan SSH
git clone git@github.com:yourusername/kedir-navigasi.git

# Navigate ke project directory
cd kedir-navigasi
\`\`\`

#### 2. Install Dependencies

\`\`\`bash
# Menggunakan npm
npm install

# Atau menggunakan yarn
yarn install

# Atau menggunakan pnpm (faster alternative)
pnpm install
\`\`\`

#### 3. Setup Environment Variables

Buat file `.env.local` di root directory:

\`\`\`bash
# Google Maps API
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here

# Gemini AI API (optional, untuk content generation)
GEMINI_API_KEY=your_gemini_api_key_here

# Analytics (optional)
NEXT_PUBLIC_GA_ID=your_google_analytics_id

# Site Configuration
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_APP_NAME=Kedir Navigasi
\`\`\`

#### 4. Run Development Server

\`\`\`bash
# Menggunakan npm
npm run start

# Atau menggunakan yarn
yarn start

# Server akan berjalan di http://localhost:3000
\`\`\`

#### 5. Build untuk Production

\`\`\`bash
# Menggunakan npm
npm run build

# Kemudian jalankan production server
npm run start

# Atau menggunakan yarn
yarn build
yarn start
\`\`\`

### Deployment

#### Deploy ke Vercel (Recommended)

\`\`\`bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel

# Deploy ke production
vercel --prod
\`\`\`

#### Deploy ke Other Platforms

**Netlify:**
\`\`\`bash
npm run build
# Upload dist folder ke Netlify
\`\`\`

**Traditional Server:**
\`\`\`bash
npm run build
# Copy build output ke server web root

### Untuk End User

#### Navigasi Website

1. **Homepage** - Lihat overview destinasi populer dan culinary highlights
2. **Menu Wisata** - Jelajahi semua destinasi dengan filter dan sort options
3. **Menu Kuliner** - Discover makanan khas Kediri dengan rating dan review
4. **Menu Budaya** - Pelajari sejarah dan budaya Kota Kediri
5. **Menu Peta** - Gunakan peta interaktif untuk navigasi real-time
6. **Menu Tentang** - Informasi tentang website dan tim
7. **Menu Kontak** - Hubungi kami dengan pertanyaan atau saran

#### Fitur Peta Interaktif

**Cara Menggunakan:**
1. Buka halaman "Peta" dari menu utama
2. Zoom in/out untuk melihat area yang berbeda
3. Klik marker untuk melihat informasi destinasi
4. Gunakan search untuk mencari destinasi spesifik
5. Filter by category untuk jenis wisata tertentu
6. Click "Directions" untuk mendapat navigasi
7. Street View tersedia dengan klik icon kamera

**Tips:**
- Enable GPS untuk akurasi lokasi terbaik
- Bookmark destinasi favorit untuk akses cepat
- Share lokasi dengan teman via WhatsApp

#### Mencari Destinasi

**Method 1 - Browse by Category:**
1. Pilih kategori di Wisata: Alam, Sejarah, Keluarga, Religi
2. Lihat list destinasi dengan rating dan jarak
3. Klik untuk melihat detail lengkap

**Method 2 - Search:**
1. Gunakan search bar di navigation
2. Ketik nama destinasi atau kata kunci
3. Tekan enter atau klik tombol search
4. Review hasil dan pilih destinasi

**Method 3 - By Map:**
1. Buka halaman Peta
2. Explore area visual dengan zoom/pan
3. Klik marker untuk info
4. Get directions

#### Mencari Kuliner

**Browse Makanan:**
1. Go to Kuliner menu
2. Pilih kategori (Makanan, Minuman, Jajanan, Kue)
3. Filter by price range (Murah, Sedang, Tinggi)
4. Sort by rating atau popularity

**Cari Warung/Restoran:**
1. Search nama makanan atau restoran
2. Lihat lokasi penjual terpercaya
3. Check rating dan ulasan
4. Click untuk get directions

#### Mobile Experience

Aplikasi ini adalah PWA dan dapat di-install:

**iOS:**
1. Buka di Safari
2. Tap Share icon
3. Select "Add to Home Screen"
4. Tap "Add"

**Android:**
1. Buka di Chrome/Firefox
2. Tap Menu icon (3 dots)
3. Select "Install app"
4. Tap "Install"

Setelah install, bisa digunakan offline!

## â“ FAQ

### Pertanyaan Umum

**Q: Apakah aplikasi ini gratis?**
A: Ya, Kediri Navigasi sepenuhnya gratis untuk semua pengguna. Tidak ada biaya tersembunyi atau subscription.

**Q: Bagaimana cara menggunakan peta interaktif?**
A: Buka halaman "Peta", gunakan zoom untuk navigate area, klik marker untuk melihat detail destinasi. Enable GPS untuk pengalaman optimal.

**Q: Apakah informasi selalu up-to-date?**
A: Kami berusaha update berkala. Untuk informasi dinamis (jam operasional, harga), silakan hubungi langsung tempat tersebut.

**Q: Bisa offline tidak?**
A: Ya! Install sebagai PWA di smartphone, dan akses offline dengan data yang sudah ter-cache.

**Q: Bagaimana cara berbagi destinasi ke teman?**
A: Setiap destinasi memiliki tombol share. Share link langsung ke WhatsApp, Facebook, atau messaging app lainnya.

**Q: Apakah ada bahasa Inggris?**
A: Saat ini hanya Bahasa Indonesia. Kami sedang mengembangkan multi-language support untuk versi mendatang.

**Q: Bagaimana cara menambah destinasi?**
A: Hubungi kami via kontak form atau WhatsApp dengan detail lengkap destinasi yang ingin ditambahkan.

**Q: Apakah ada API publik yang bisa digunakan?**
A: Saat ini API belum publik. Silakan hubungi tim untuk kebutuhan integrasi khusus.

**Q: Bagaimana keamanan data saya?**
A: Semua komunikasi ter-enkripsi HTTPS. Data tidak disimpan di server, hanya di perangkat Anda. Privacy policy tersedia di halaman Kontak.

**Q: Bisa digunakan di aplikasi pihak ketiga?**
A: Ya, dengan permission khusus. Hubungi tim untuk API access request.

**Q: Siapa yang mengoperasikan website ini?**
A: Kedir Navigasi dioperasikan oleh Tim Pengembang profesional bekerja sama dengan Dinas Pariwisata Kota Kediri.
